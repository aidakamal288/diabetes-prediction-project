# Diabetes Prediction Project

## Overview
This project aims to predict whether a person is diabetic or non-diabetic using Machine Learning techniques and health-related indicators.

## Dataset
BRFSS 2015 Diabetes Health Indicators Dataset

- Rows: 253,680
- Columns: 22

## Machine Learning Models
- Logistic Regression
- Random Forest
- XGBoost

## Best Model
XGBoost achieved the best overall performance with:
- Accuracy: 84.75%
- Precision: 54.05%
- Recall: 7.91%

## Features Used
- BMI
- Age
- HighBP
- HighChol
- PhysActivity
- LifestyleScore

## Deployment
The model was deployed using Streamlit.
